int* bresenham_line_algorithm ( int  x0,
int  y0,
int  xf,
int  yf 
)

Generates the Player_projectile's array of points.

Uses bresenham's algorithm, and returns the points in an array with the format [x0, y0, x1, y1, x2, y2, x3, .....] code apropriated from http://members.chello.at/~easyfilter/bresenham.html

Parameters
x0The first point's x value
y0The first point's y value
xfThe target point's x value
yfThe target point's y value
Returns
Pointer to the generated array, of size 2*PROJECTILE_SIZE

Checks for player projectiles colisions, and destroys the objects involved.

Checks for colisions between the player projectiles and enemies, and checks if the player projectiles have gone out of bounds. If those conditions are met, the player projectiles are destroyed, and so are the enemy objects, in the former case
